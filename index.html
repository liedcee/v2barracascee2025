📋 Configuração Necessária

Para usar este sistema, você precisa configurar uma planilha do Google:

  1. Acesse Google Sheets e crie uma nova planilha
  2. Nomeie a primeira aba como "Inscricoes"
  3. Crie os cabeçalhos na linha 1: Nome | Turma | Barraca | Turno | Data
  4. Vá em Extensões → Apps Script
  5. Cole o código do script (fornecido separadamente)
  6. Publique como Web App
  7. Cole a URL do Web App no campo abaixo
